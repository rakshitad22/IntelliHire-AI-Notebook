# IntelliHire AI – Resume Screening and Recruitment Analytics System

This project demonstrates AI-powered resume screening and recruitment analytics using:

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- TF-IDF
- Cosine Similarity
- WordCloud
- Jupyter Notebook

## Features

- ATS Score Calculation
- Resume Analytics
- Skill Frequency Analysis
- Confusion Matrix
- Candidate Ranking
- KPI Dashboard
- WordCloud Visualization
- Recruitment Insights
